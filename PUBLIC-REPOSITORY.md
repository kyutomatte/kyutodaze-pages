# Public Deployment Repository

This repository is the public deployment copy of the KYUTO.MATTE portfolio.
It intentionally starts with a new Git history and does not contain the
private source repository's `docs/superpowers/` directory.

The GitHub Pages workflow publishes the site at the repository project URL.
The Formspree endpoint in the workflow is public by design; do not add Gmail,
SMTP, Formspree management, or other credentials to this repository.
